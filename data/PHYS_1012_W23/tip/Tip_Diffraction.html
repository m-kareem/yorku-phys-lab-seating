Coax cable is permenantly attached to the scope. Please ignore the request in the Manual which says to put it aside. All coax and fiber cable lengths can be found on the cables' labels except short coax cables The length for the short cable is given in the journal template. Use PS_OUTPUT software (in the Speed of light folder) to enable/disable power supply output. In case of getting red indicator on the software front panel, quit and run it for a second time. Please use yellow button at the bottom to quit the program at the end of lab to make sure that the power supply output is disabled.
